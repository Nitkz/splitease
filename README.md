# 💸 SplitEase

SplitEase is a simple, beautiful, and intuitive Single Page Application (SPA) designed to help groups track and split expenses seamlessly.

## ✨ Features
- **Member Management**: Easily add and manage people involved in expenses.
- **Expense Logging**: Record who paid for what and split it among specific members or everyone.
- **Real-time Dashboards**: View total spending and individual balances instantly.
- **Smart Settlements**: Uses a greedy algorithm to calculate the most efficient way for everyone to settle their debts (who owes whom and how much).

## 🚀 Tech Stack
- **HTML5** & **CSS3** (Clean, modern aesthetic using CSS variables and the Inter font)
- **Vanilla JavaScript** (ES6+)
- **Zero Dependencies**: No frameworks or build tools required.

## 🛠️ How to Use
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/Nitkz/splitease.git
   ```
2. Open `index.html` directly in your web browser.
3. Start adding members and tracking expenses!
